* Renaming a linux user

- Log in as a different priviledged user, either root or someone with sudo access
- Kill all processes running under user to be renamed

  #+begin_src bash
    sudo pkill -u old_username
  #+end_src

- Rename the username and home directory

  #+begin_src bash
    sudo usermod -l new_username -d /home/new_username -m old_username
  #+end_src

- Rename the user's primary group

  #+begin_src bash
    sudo groupmod -n new_username old_username
  #+end_src

- Update Sudoers: If the old username was explicitly named in the /etc/sudoers file, run sudo visudo to update the name.
- Fix Configurations: Application settings, internal scripts, or paths hardcoded to /home/old_username must be manually updated to point to the new directory
